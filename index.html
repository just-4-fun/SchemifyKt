Schemify.Kt

Schemification and serialization API for Kotlin

Download this project as a .zip file Download this project as a tar.gz file

APIs:

  • Schemification API: Creating a scheme of a class.
  • Serialization API: Transforming an instance of schemified class from / to other format.

Schemification API:

Structure of a class can be represented as a scheme by mapping class fields to scheme Props. I.e. scheme here is representation of class state as set of props. Example: 1.pdf
